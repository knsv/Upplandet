# Chapter 4: Talents (PDF s. 72–82)
## I korthet
Talents är återanvändbara specialregler som gör karaktärer distinkta: nya actions, förbättrade manövrer, passiva fördelar eller regelundantag. Genesys organiserar talents i **tiers** (1–5) för ungefärlig kraftnivå.

## Talent-typer (hur de används)
- **Passiva**: “alltid på” eller triggar i tydliga situationer.
- **Aktiva**: kräver ofta **action** eller **maneuver**, ibland kostnad i Strain eller Story Point.
- **Reaktioner/utlösta**: kan trigga på Advantage/Triumph eller särskilda händelser.
- **Ranked talents**: kan köpas flera gånger.

## Tiers (balanssignal)
- **Tier 1**: breda, ofta små men frekventa förbättringar.
- **Tier 2–3**: tydligare nisch och effekt; börjar forma spelstil.
- **Tier 4–5**: kraftfulla eller kampanjdefinierande effekter; kräver ofta mer investering/prereqs.
