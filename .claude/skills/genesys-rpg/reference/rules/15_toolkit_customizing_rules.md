# Chapter 1: Customizing Rules (PDF s. 191–204)
## I korthet
Guidelines för att skapa eget Genesys-innehåll: skills, archetypes/species, talents, items och adversaries. Tanken är att återanvända systemets “valutor” (dice upgrades, Boost/Setback, strain, qualities) för balans.

## Skapa
- **Skill**: tydlig nisch, minimal överlapp; rimlig characteristic-koppling.
- **Archetype/species**: styrkor + begränsningar; jämför mot basprofiler.
- **Talent**: trigger + effekt + kostnad; sätt tier efter bredd/styrka.
- **Item**: statblock + qualities + rarity; bestäm hard points om ni använder attachments.
- **Adversary**: koncept → typ → roll → stats → taktik.
