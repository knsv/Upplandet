# Chapter 3: Build an Adventure (PDF s. 237–241)
## I korthet
Ett recept för att bygga äventyr: bestäm ton, formulera story concept, sätt mål och insatser, skapa NPC:er och platser, och bryt ner i encounters.

## Checklista
- Tone → Concept → Goals
- Centrala NPC:er med motivation
- Platser/scener med valpunkter
- Encounters som kan svänga via Advantage/Threat
